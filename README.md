# MUIN AI COO 랜딩페이지

AI가 COO 역할을 수행하는 서비스의 랜딩페이지입니다.

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS 4
- TypeScript
- Vercel 배포

## 개발

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # 프로덕션 빌드
```

## 배포

Vercel에서 GitHub 레포 연결 후 자동 배포됩니다.

## 구조

```
src/app/
├── globals.css   # Tailwind + Pretendard 폰트
├── layout.tsx    # 메타데이터, OG 태그
└── page.tsx      # 전체 랜딩페이지
```
